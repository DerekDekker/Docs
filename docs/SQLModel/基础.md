## 数据库
??? abstract "代码"

    ```python title="database.py"
    from sqlmodel import SQLModel, create_engine
    
    # 连接数据库
    engine = create_engine("sqlite:///database.db")
    
    # 创建数据库
    def create_db_and_tables():
        SQLModel.metadata.create_all(engine)
    ```
