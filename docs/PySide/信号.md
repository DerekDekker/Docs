## 基础

参数可以是 组件实例

```python
组件.信号.clicked.connect(槽函数)

# 传递参数
组件.信号.clicked.connect(lambda: 槽函数(参数))

# 信号绑定信号
信号.connect(信号)

# 信号绑定信号 for遍历信号时使用
组件.信号.clicked.connect(functools.partial(槽函数, 参数))
```
