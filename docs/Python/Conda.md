# Conda

Python 的虚拟环境

---

## 命令


`source ~/.bashrc`  激活环境



`conda env list`  所有环境


`conda install 包`  安装包


`conda create -n 名称 python=3.5`   创建虚拟环境


`conda activate 名称`  开启xxxx环境


`conda deactivate`  关闭环境            


`conda remove -n 名称 --all`  删除环境


`conda create -n example --offline`  断网创建环境
