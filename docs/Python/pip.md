# pip

安装Python模块

[官网](https://pypi.org)

Debian下安装
sudo apt-get install python3-pip

---

## 语法

`pip install 组件名`         安装组件

`pip install -U 组件名`    升级组件

`pip uninstall 组件名`      卸载组件

`pip list`                          查看本地组件


`pip freeze > requirements.txt`

`pip install -r requirements.txt`


---

## 扩展

`-i https://pypi.douban.com/simple`    镜像源

`-r xxx.txt`    安装文本里的包
