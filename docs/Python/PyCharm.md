### 快捷键

!!! note

    Ctrl+F					搜索

    Shift+Ctrl+F				全局搜索

    Shift+Ctrl+R				全局替换

    两下Shift					搜索

    Ctrl+R					搜索替换

    Ctrl+/					注释

    Ctrl+y					删除一行

    Ctrl+d					复制一行

    Shift+回车					换行

    Shift+Tab					恢复缩进
    
    Shift+Ctrl++				全部展开

    Ctrl++					展开

    Shift+Ctrl+-				全部收起

    Ctrl+-					收起
