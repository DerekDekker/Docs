# 样式组件

---
## TextStyle

```dart
TextStyle(   // 文本样式
    fontSize: 30,
    color: Colors.red,  // 文本颜色
    fontWeight: FontWeight.w500,  // 字体粗细
    fontStyle: FontStyle.italic,  // 斜体 字体样式
    fontFamily: 'XiaoWei',  // 自定义字体
)
```
