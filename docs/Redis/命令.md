安装
apt install redis-server

是否正常运行
systemctl status redis-server

配置文件
/etc/redis/redis.conf

# Redis 客户端：
redis-cli
    -p 6379       端口
    -a test123    密码

----------------------------------------------------------------------------redis命令
> 查询密码
config get requirepass

> 设置密码
config set requirepass test123
