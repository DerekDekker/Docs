## axios

[官网](https://axios-http.com)

```javascript
<script>
import axios from 'axios'
</script>
```

`可以选择写在 mounted(){} 内`

```javascript
axios({
     method: "get",
     url: "API地址",
 }).then(res=>{
     // 赋予给变量
     this.chengpin = res.data;
 })
```
