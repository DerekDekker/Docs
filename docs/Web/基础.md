```javascript

<input type="text" v-model.trim="username">
```

## 事件

```javascript
<button v-on:click="count+=1">点击</button>
<button @click="count+=1">点击</button>
<button @click="函数名称">点击</button>
<button @click="函数名称(参数)">点击</button>
```

| 显示方式 | 效果         |
| -------- | ------------ | 
| click        | 点击事件 |

??? 接受参数

    函数名称($event) 事件中可以接受参数 参数为点击的按钮 可以在事件中如修改按钮文本 

### 事件修饰符



```javascript
阻止默认事件
<a @click.prevent="函数" href="https://google.com">Google</a>

阻止事件冒泡
<a @click.stop="函数" href="https://google.com">Google</a>

````

```javascript
阻止默认事件
函数(e){
    e.preventDefault();
}

阻止事件冒泡
函数(e){
    e.stopPropagation();
}
```

## 插槽

将父级HTML部分显示在子组件中

```javascript
// 父组件
<template>
    <子组件名称>HTML部分</子组件名称>
</template>


// 子组件
<template>
    // 显示在此处, 没有收到数据可以填写插槽默认值
    <slot>默认值</slot>
</template>
```

具名插槽

```javascript
// 父组件
<template>
    <子组件名称>
        <template v-slot:插槽名称1>HTML部分</template>
        <template v-slot:插槽名称2>HTML部分</template>
        <template #插槽名称3>HTML部分</template>
    </子组件名称>
</template>



// 子组件
<template>
    <slot name="插槽名称1">默认值</slot>
    <slot name="插槽名称2">默认值</slot>
</template>
```

子组件传递参数给父组件

```javascript
// 父组件
<template>
    <子组件名称 v-slot="slotProps">
        {{ slotProps.属性名 }}
    </子组件名称>
    // 具名插槽
    <子组件名称>
        <template #插槽名称3="slotProps">{{ slotProps.属性名 }}</template>
    </子组件名称>
</template>

// 子组件
<template>
    <slot :属性名="值">默认值</slot>
</template>
```

## 动态组件

将组件赋值给一个变量 将变量作为组件显示
```javascript
<component :is="组件变量"></component>

// 保持存活
<keep-alive>
    <component :is="组件变量"></component>
</keep-alive>
```
