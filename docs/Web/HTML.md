# HTML

---
## 基础

```html
<html>
    <head></head>
    <body></body>
</html>
```

---
## 标签

### br

```html
<!--换行-->
<br />
```

### hr

```html
<!--水平线-->
<hr />
```

### img

```html
<!--图片-->
<img src="地址" width="100" height="100" alt="皮卡丘" />
```

### a

```html
<a href="地址"></a>
<!--新窗口打开-->
<a href="地址" target=_blank title="链接提示"></a>
```

```css
{
    text-decoration: none;  /*去掉下划线*/
}
```

---
##  文本

```html
<!--标题-->
<h1></h1>
<h6></h6>
<!--粗体字-->
<b>...</b>
<!--斜体字-->
<i>...</i>
<!--底线-->
<u>...</u>
<!--段落-->
<p></p>
```

### article

```html
<article></article>
```

#### footer

```html
<footer>
    <h1>这里是文章标题</h1>
    <span>作者：张三</span>
    <time datetime="2023-06-15T16:06:20.849Z">发布时间：2023年6月15日</time>
</footer>
```
