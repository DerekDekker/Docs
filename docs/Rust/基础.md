# Rust 基础

---
## 命令

```shell
# 编译
rustc 文件.rs
```

---
## 基本

```Rust
# 入口
fn main (){
    println!("Holle Word!")
}
```

```Rust
# 定义函数
fn 函数名 () {
    //
}
```

### 变量定义

```Rust
let 变量名 = 值;
let 变量名:数据类型 = 值;

// 可变变量
let mut 变量名 = 值;
```




---
## 语法

```Rust
# 输出
println!("Holle Word!")
println!("Holle Word!{}", 变量名)
```
