# Rust 基础



[库](https://crates.io)

---
## 命令

```shell
# 编译
rustc 文件.rs
```

`rustup update` 更新

`rustc 文件.rs`  生成二进制可执行文件

`cargo new 项目名`  创建项目

`cargo buile`  构建项目 生成二进制可执行文件

`cargo buile --release`  生成环境

## 基本

```Rust
# 入口
fn main (){
    println!("Holle Word!")
}
```

```Rust
# 定义函数
fn 函数名 () {
    //
}
```

### 变量定义

```Rust
let 变量名 = 值;
let 变量名:数据类型 = 值;

// 可变变量 
let mut 变量名 = 值;

// 常量 字母全部大写 块级作用域
const 变量名 = 值;

// 静态变量 字母全部大写 生命周期是整个程序的生命周期 
static 变量名 = 值;

```

### 命名空间

命名空间 作用域 内声明的变量与获取的变量是同一个变量, 命名空间结束后命名空间内的变量自动销毁

```Rust
{
    // 代码块
}
```

## 语法

```Rust
# 输出
println!("Holle Word!")
println!("Holle Word!{}", 变量名)
println!("Holle Word!{变量名}")
```
