## 整型

| 类型   | 字节 | 范围                                         |
|------|----|--------------------------------------------|
| i8   | 1  | -128 到 127                                 |
| i16  | 2  | -32768 到 32767                             |
| i32  | 4  | -2147483648 到 2147483647                   |
| i64  | 8  | -9223372036854775808 到 9223372036854775807 |
| i128 | 16 | (2的129次方)/2                                |
| u8   | 1  | 0 到 255                                    |
| u16  | 2  | 0 到 65535                                  |
| u32  | 4  | 0 到 4294967295                             |
| u64  | 8  | 0 到 18446744073709551615                   |
| u128 | 16 | 2的129次方                                    |
