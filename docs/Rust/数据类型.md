## 

数组

元素必须相同

```Rust
// 定义
let 名称 = [值, 值, 值];
let mut 名称 = [值, 值, 值];
```

`数组[0]`  获取元素

`数组[0] = 值`  赋值 元素

`数组.len()`  获取长度 


##

元组

元素可以不同

```Rust
// 定义
let 名称 = (值, 值, 值);
let mut 名称 = (值, 值, 值);
```

`元组.0`  获取元素

`元组.0 = 值`  赋值 元素
